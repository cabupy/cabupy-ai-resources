# Ejemplo: Nueva pagina de administracion

## Entrada

- Crear la pagina `/admin/categorias` con listado, busqueda y formulario.

## Salida esperada

- Ruta nueva integrada al sistema de navegacion del proyecto.
- Carga de datos usando la capa de server state definida por el proyecto.
- Formulario validado con esquema y manejo de estados de error.
- Estados de loading, empty y success cubiertos en UI.
