# Ejemplo: Componente reutilizable

## Entrada

- Crear un selector reutilizable de miembro para usar en varios modales.

## Salida esperada

- Componente reutilizable con busqueda y tipado estricto.
- Integracion compatible con formularios y validacion del proyecto.
- API del componente documentada (props, estados y eventos).
- Comportamiento accesible con teclado y foco visible.
